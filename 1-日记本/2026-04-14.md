## 📋 脚本功能总览

这是一个 **macOS 视频文件管理自动化脚本**，主要用于批量处理、整理、压缩和优化视频文件，特别适合管理直播录像、短视频等内容。

---

## 🎯 核心功能模块

### 1. **目录管理与整理**

表格：

| 功能           | 说明                                              |
| :----------- | :---------------------------------------------- |
| **自动移动视频**   | 根据文件名中的关键词（如 `用户名-视频.mp4`）自动创建 `DY-用户名` 目录并移动文件 |
| **合并同名目录**   | 自动合并名称相同但带不同大小标签的目录                             |
| **更新目录大小标签** | 自动计算目录大小并添加到目录名（如 `DY-用户（1.2G）`）                |
| **空目录管理**    | 自动隐藏空目录，或显示所有目录                                 |

### 2. **视频压缩与转码 (`-compress`)**

- **质量压缩**：使用 H.265/HEVC 编码，保持清晰度的同时大幅减小体积
    
- **分辨率缩放**：支持 `4k` / `2k` / `1080p` / `720p` 等预设
    
- **视频截取**：支持按时间段截取（`-ss` / `-to` / `-t`）
    
- **批量处理**：支持通配符匹配和递归处理
    
- **自动检测**：自动识别并去除视频末尾的"直播已结束"画面
    

### 3. **OCR 智能重命名 (`-ocr`)**

- **自动识别用户名**：从视频前1秒提取画面，使用 EasyOCR 识别用户名
    
- **交互式选择**：多个候选结果时，可用方向键选择
    
- **分片视频处理**：智能识别 `_part_00` 等分片文件，统一重命名
    
- **失败处理**：未识别成功的文件自动移动到"未成功分类"目录
    

### 4. **视频完整性检测 (`-check`)**

- **"直播已结束"检测**：动态加速采样 + OCR 识别，检测视频末尾的结束画面
    
- **画面静止检测**：检测视频是否提前结束（画面不再变化）
    
- **自动标记/移动**：问题视频自动标记或移至"待优化视频"目录
    
- **并行处理**：支持多线程并行检测（`-j` 参数）
    

### 5. **视频画面处理**

表格

|功能|说明|
|:--|:--|
|**裁剪 (`-crop`)**|支持手动指定区域或快捷预设（下半部分、左半部分等）|
|**有效内容截取 (`-trim`)**|自动检测并去除"直播已结束"画面，支持快速模式/重编码模式|
|**时长截取 (`-cut`)**|从开头截取指定时长的视频|
|**1小时切分 (`-split1h`)**|将超过1.5小时的视频按小时无损切分|
|**时间戳修复 (`-fix-timestamp`)**|修复视频时间戳偏移问题|

### 6. **标签智能分类 (`-tag`)**

- 读取视频的 **Finder 彩色标签**
    
- 根据标签自动匹配目标目录中的 `【】` 文件夹
    
- 移动到对应的 `DY-关键词` 子目录
    
- 可选创建符号链接视图 `DY-分类浏览`
    

### 7. **文件夹图标设置 (`-icon` / `-rmicon`)**

- 自动提取视频第一帧作为文件夹图标
    
- 生成 512x512 的高清图标
    
- 支持递归批量设置或移除
    

### 8. **系统健康度检查**

- 处理前自动检查 CPU、内存、磁盘使用率
    
- 系统负载过高时提醒用户
    
- 支持用户确认是否继续处理

### 关键技巧

- **递归处理**：支持 `-r` 无限递归或 `-d <层数>` 指定深度
    
- **并行处理**：`-j` 参数支持多线程加速
    
- **文件名解析**：使用正则提取关键词、处理大小标签
    
- **临时文件管理**：使用 `mktemp` 确保清理
    
- **错误处理**：详细的返回值和错误提示

## 📖 典型使用场景

### 场景1：整理下载的直播视频

bash

复制

```bash
# 1. 自动按用户名分类
./rename_with_size.sh /path/to/videos

# 2. OCR 识别重命名（添加中文用户名前缀）
./rename_with_size.sh -ocr /path/to/videos

# 3. 检测并去除"直播已结束"画面
./rename_with_size.sh -trim /path/to/videos

# 4. 压缩视频节省空间
./rename_with_size.sh -compress *.mp4 quality
```

### 场景2：批量压缩转码

bash

复制

```bash
# 递归压缩所有 MP4 为 1080p
./rename_with_size.sh -r -compress '*.MP4' 1080p

# 批量压缩并移动到指定目录
./rename_with_size.sh -r -compress '*.MP4' 4k /path/to/output
```

### 场景3：按标签自动分类

bash

复制

```bash
# 根据 Finder 标签自动分类，并创建符号链接视图
./rename_with_size.sh -tag /source /target -link
```

1. **Bash 高级技巧**：关联数组、函数返回值、进程替换、信号处理
    
2. **视频处理流程**：ffmpeg 参数组合、关键帧定位、无损/重编码选择
    
3. **OCR 集成**：Python 与 Bash 的混合编程，批量处理优化
    
4. **macOS 特性**：Finder 标签、文件夹图标、chflags 隐藏属性
    
5. **工程化思维**：模块化设计、错误处理、用户交互、性能优化
    

这个脚本是一个很好的 **"Bash + 多媒体处理 + 自动化工作流"** 学习范例，代码量虽大但结构清晰，适合作为复杂 Bash 项目的参考模板。
